## Installation

```bash
mix escript.install github ajfAfg/HaLLoween
```

### Note

This program will be placed in `~/.mix/escripts`.
So, if you haven't passed the path `~/.mix/escripts` yet, please do so as follows.

```bash
export PATH=$PATH:~/.mix/escripts
```
